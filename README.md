# Tandem Trivia

## Using
- NextJS(ReactJS)
- Firebase
- ChakraUI

## How to use
1. Copy .env.development.local.copy to .env.development.local and fill in Firebase configuration variables
```
cp .env.development.local.copy .env.development.local
```
2. Install NPM packages
```
npm install
```
3. Running on local
```
npm run dev
```

http://localhost:3000/
